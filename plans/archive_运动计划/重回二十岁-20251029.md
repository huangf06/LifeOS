⏺ 三年身体重建计划 (2025-2028)

  🎯 核心理念

  不是短跑，而是马拉松
  - 优先级：健康 > 习惯 > 指标
  - 原则：可持续 > 快速见效
  - 重点：不受伤 > 突破极限

  🏋️ 第一年详细计划 (立即启动)

  Phase 1: 身体唤醒期 (第1-2个月)

  目标：重建神经肌肉连接，避免受伤

  每周训练4次：

  周一：上肢启动
  - 靠墙静蹲：3组 x 30秒
  - 俯卧撑（跪姿/斜上）：3组 x 8-10个
  - 弹力带划船：3组 x 12个
  - 平板支撑：3组 x 30秒

  周三：下肢+核心
  - 深蹲（徒手）：3组 x 15个
  - 箭步蹲：3组 x 10个/腿
  - 臀桥：3组 x 15个
  - 死虫：3组 x 10个

  周五：上肢力量
  - 辅助引体（弹力带）：4组 x 最大次数
  - 卧推（空杆或轻重量）：4组 x 8个
  - 肩推：3组 x 10个
  - 吊环支撑：3组 x 15秒

  周六/日：有氧恢复
  - 快走/慢跑：20-30分钟
  - 配速：能聊天的强度
  - 或骑车、游泳

  第1-2个月关键指标：
  - 完成率 > 90%（比强度更重要）
  - 无疼痛（正常肌肉酸痛除外）
  - 睡眠质量改善
  - 体重控制在合理范围

  ---
  Phase 2: 基础建设期 (第3-6个月)

  逐步加入跑步和力量进阶

  周一：力量A（推）
  - 卧推：4组 x 6-8个（渐进负重）
  - 上斜哑铃推：3组 x 10个
  - 三头训练

  周二：跑步
  - 轻松跑：3-5km，配速6:30-7:00/km

  周四：力量B（拉）
  - 引体向上（辅助递减）：5组 x 最大
  - 杠铃划船：4组 x 8个
  - 二头训练

  周五：间歇跑
  - 400米 x 5组，配速5:00/km，休息2分钟

  周六：长距离
  - 8-10km，配速6:30-7:00/km

  周日：休息或瑜伽/拉伸

  ---
  🔧 配套系统

  1. 伤病预防系统

  每次训练前：
  - 动态热身：10分钟
  - 关节活动：肩、髋、踝

  每次训练后：
  - 静态拉伸：15分钟
  - 泡沫轴放松

  每周必做：
  - 1次瑜伽/柔韧训练
  - 1次理疗/按摩（如需要）

  警报信号：
  - 关节疼痛 → 立即停止
  - 持续酸痛 > 3天 → 休息
  - 睡眠质量下降 → 减量

  2. 恢复系统

  睡眠：7-9小时/天
  营养：
  - 蛋白质：120-140g/天（体重70kg）
  - 水：3L/天
  - 补剂：根据需要（肌酸、蛋白粉）

  主动恢复：
  - 轻度有氧：散步、游泳
  - 按摩、冷热浴

  3. 进度追踪系统

  每周记录：
  - 训练完成率
  - 身体感受（1-10分）
  - 睡眠质量
  - 体重/体脂

  每月测试：
  - 引体向上最大次数
  - 3km跑时间
  - 卧推1RM

  每季度评估：
  - 调整训练计划
  - 拍照对比
  - 体检指标
